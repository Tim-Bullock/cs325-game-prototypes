Laundromat Escape!

  • Originality: Laundromat Escape is an original game that mixes the two thematic cards of laundromat and a person with many cats. As far as I know, there is no other game like it where you must escape a laundry map in order to avoid cats. It is such a unique premise.
  • Technical Merit: This game uses many new technical aspects of phaser that I have not used before including spriteset animations, collision handlers, and dedicated game scenes for loading, winning/losing, and game play screens.
  • Prototype Postmortem: I learned lots by making this prototype, by far the most significant thing that I learned from this prototype was how to use scenes to create a more cohesive game experience that guides the user throughout the game and make the game more cohesive.

  • High Concept: In this challenging game someone with several cats has brought them all to the laundromat, and now you must escape the laundromat while avoiding the cats so that you clean nice laundry does not get messed up.
  • Theme: For this assignment I chose the storymatic cards "Person with a lot of cats" and "Laundromat". It was interesting pairing such different themes.
  • Mandated Variety: For the mandated variety aspects I chose:
    • Inputs: Clickable Buttons and Keyboard
    • Randomness: A fair amount of randomness (speed of cat sprites)
    • Playstles: Achiever and Performer
    • Genres: Overhead
  • Prototype Goal: In this prototype, I wanted to test the core game mechanics such as the ability for the user to move when the cat's speeds were randomized, as well as have a deeper dive into the stylistic aspects game creation in phaser to see how they effect the user experience.
  • Player Experience Goals: While playing this game users should feel challenged by the randomness of the cat's movements and puzzled by the strange premise of the game.
  • Gameplay: In this prototype the user has five simple actions/controls that are needed to find play the game:
    • Mouse: Navigate Menu's/Buttons
    • Left Arrow Key: Run Left
    • Right Arrow Key: Run Right
    • Up Arrow Key: Run Up
    • Down Arrow Key: Run Down
    To play this game the user must use the arrow keys in order to reach the exit of the Laundromat (located at the bottom of the screen). However there are several cats that get in the way; if you run into one, you lose and must start over. The cats mover back and forth at random velocities which makes it challenging to predict when it will be optimal to move. In future prototypes of this game, I would like to add more levels (maybe even randomly generated ones, add the ability for the cats to "jump" over the washing machines to make the game more challenging, and give the cats and animated running spriteset.
  • Strategies: The spaces in between the rows washers/dryers are useful as you can pause here and wait for the opportune moment to get past the cats.
  • Story/Setting/Premise: The premise of this game is that one day a crazy stranger turns up to the laundromat will all of her cats. This is not just one or two! There are many! You, the player, were just finishing your laundry and now must find a way to escape the laundromat without running into any cats as they will claw and tear your clothes to shreds. This tension gives rise to the game tension as it is the players sole goal to try to exit the laudromat.
  • Target Audience: People who like cats, top-down games where you must avoid obstacles (such as classic Frogger Arcade game), or intriguing games with strange premises.
  • Play Time: Less than one minute if you are good/lucky, up to five minutes otherwise.
