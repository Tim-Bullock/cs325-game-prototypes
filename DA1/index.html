Space Savior

  • Originality: This is a simple yet interesting adventure-like/ puzzle game that requires the user to find an astronaut that is lost in space. To my knowledge, there is no other game that is like it, but it is similar to maze games where the user has to find an object or location.
  • Technical Merit: This simple game uses many features of phaser. For example the asteroids are placed in the map using a tile map, and the rocket's movements are set in such a way that they represent the way a rocket would be able to travel in space using angular velocities and drag. The background moves by using a tilesprite which changes based on the rockets location.
